Penjelasan Singkat Mengenai Core Pada Processor

Penjelasan Singkat Mengenai Core

Processor merupakan komponen terpenting pada sebuah komputer. Processor adalah sebagai otak yang menjalankan proses dan pengendali kerja komputer & bekerjasama dengan perangkat komputer lainnya. Di satu sisi Processor memiliki satuan yaitu Mhz atau Ghz semakin besar kecepatan suatu Processor maka semakin cepat kinerja komputer saat melakukan proses pengeksekusian data.

Fungsi processor pada komputer mempunyai fungsi yang sangat penting . Pada umumnya fungsi dari sebuah Processor adalah untuk mengeksekusi data yang telah diterima dari masukkan atau input, kemudian akan menghasilkan pengeluaran atau output.

– Thread pada Processor

Thread merupakan kumpulan instruksi atau proses yang dilakukan secara independen.

– Multithreading pada Processor

Multithreading merupakan kemampuan yang memungkinkan beberapa kumpulan proses dilakukan secara bersamaan dalam suatu program. Multithreading bermanfaat untuk membuat intruksi yang interaktif, contohnya pada game. Program tetap dapat menggerakkan beberapa objek sambil memberi kesempatan pengguna melakukan respon melalui mouse atau keyboard.

– Core pada Processor

Core merupakan suatu unit processing yang membaca semua proses untuk mengeksekusikan tindakan tertentu. Instruksi ditampung ketika dijalankan secara real time. Sederhananya, semua yang teman-teman lakukan pada komputer harus diproses oleh processor. Setiap membuka aplikasi, mengetik, dsb. semua itu membutuhkan processor untuk memproses program tersebut.

– Setiap ‘core’ pada Processor memiliki sebuah CPU (Central Processing Unit) dan merupakan bagian dari CPU yang bekerja. Contohnya, sebuah chip dual-core mungkin terlihat seperti chip single-core, namun sebenarnya memiliki 2 buah CPU yang ada di dalamnya.

– Dengan CPU tambahan, PC akan lebih cepat ketika melakukan multi-tasking, seperti browsing sambil extract file. Pada single-core proses ini dijalankan secara bersamaan dengan CPU yang sama, sedangkan pada dual-core proses ini dibagi ke masing-masing core CPU, sehingga proses akan bekerja lebih cepat .

Nah, itu dia pembahasan singkat kali ini, Tentang Core pada Processor. Semoga dapat bermanfaat untuk kalian dan Terima Kasih sudah berkunjung!
